# Blinkit Sales Dashboard

## Objective
Analyze Blinkit grocery sales data to understand outlet performance and product trends.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Insights
- Tier 1 outlets generate highest revenue
- Large outlets perform better than small outlets
- Snacks category contributes maximum sales

Dataset file is included in this repository.
